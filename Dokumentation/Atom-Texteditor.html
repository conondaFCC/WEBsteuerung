How to get the Title automatically in here?

Shortcuts / Kurzbefehle

Shortcut Erklärung / Auswirkung
hmtl Durch die Eingabe von html in einem leeren html-Dokument (*.html) wird via auto completition ein Teil eines html Dokumentes erstellt. Über Doctype, UTF-8, Title, Head bis Body. Bisher der einfachste Weg ein neue html vorlage zu erhalten.
Ctrl+Shift+p Atom Befehlssuche aufrufen um nach einem Befehl / einer Aktion zu suchen.
Ctrl+M Cursor an Anfang / Ende von Tag bewegen.
Ctrl+Shift+M Vorschau des aktuellen Dokumentes.
Ctrl+Enter

Ctrl+Shift+Enter		 Neue Zeile einfügen. Dabei wird der atuelle Tag verlassen. Klappt aber nur wenn Endtag auf gleicher Zeile wie Cursor. Inklusive Shift neue Ziele oberhalb einfügen.
Ctrl+up/down Zeile oder Block nach oben / unter verschieben.
Tagname eingeben Listet Tags auf welche mit diesen Zeichen beginnen. Mittels Auswahl der Möglichkeit (Cursor, Enter) oder Abbruch (Leertaste). Erstellt Anfangs und Endtag und setzt Cursor hinein.
table, tr, td Tags for table, tablerow, tablecell
href Tag for a Hypertext-Link. TAG
