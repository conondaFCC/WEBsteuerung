    Vorrausseztung zum verwenden von vue v-Direktiven

      Um die v-direktiven zu verwenden muss folgendes zuerst gemacht werden.
    • Code einfügen in html: 
         
  <code><script src="https://unpkg.com/vue"></script>
    • Zudem muss im html ein Tag mittels ID gekennzeichnet sein. 
      
  <div id="ID-Name">
    • Standard vue.js 
          
      var Var-Name = new Vue({
        el: '#ID-Name',
        data: {
          message: 'Hello Vue!'
        }
      })
      el steht für Element Data ist das Standardobjekt in welchem nun Attributte definiert werden können oder weitere Objekte oder Funktionen abgelegt werden.
    v-bind
    Direktive Beschreibung der Dirktive mit Beispielen
    {{ElementattributeName}} innerhalb eines div Tag mit Elements kann der Inhalt eines Element attribute in HTML anzeigen
    v-bind:attribut ="wert" Bsp. v-bind:title
